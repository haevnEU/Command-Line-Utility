Detailed Description

This class contains a progressbar for CLI.

This class will handle a progessbar. The object will only work for a command line interface. The progressbar will look like

[====================> ] (50%)

There are different visual states of the progressbar

Possible color codes
ColorDescription
GreenProgressbar finished and the operation associated should be successfull
YellowThe operation was canceled
Red

The operation failed or was aborted

WhiteDefault foreground color
BlueCustom foreground color
CyanCustom foreground color
PurpleCustom foreground color

The method required an argument as index, if the index is greater than 3 a modulo operation will be done to ensure the index is in range of the color array. There a 4 different possible colors:
